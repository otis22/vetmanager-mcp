# PRD: E2E тесты для сущностей Этапа 2.5

## Контекст

Этап 2.5 добавил MCP-инструменты для 28 новых сущностей Vetmanager API, распределённых по 5 модулям:
`tools/reference.py`, `tools/finance.py`, `tools/warehouse.py`, `tools/clinical.py`, `tools/operations.py`.

Ни одна из этих сущностей не покрыта тестами. Текущее покрытие ограничено 7 сущностями из Этапа 2.

## Цель

Покрыть все новые инструменты:
- **mock/contract-тестами** (через `respx`) — проверяют, что клиент делает правильный HTTP-запрос и корректно разбирает ответ
- **real API тестами** (против `devtr6`) — проверяют, что эндпоинты доступны и возвращают ожидаемую структуру

## Ограничения декомпозиции

Каждая подзадача ≤ 150 строк кода, ≤ 2 человеко-часов.
