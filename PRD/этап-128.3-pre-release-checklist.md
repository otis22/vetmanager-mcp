# Этап 128.3. Pre-release checklist before deploy

## Контекст

Даже без доступа к production server можно закрыть локальную часть release gate:
проверить release checklist, deploy scripts, rollback notes и убедиться, что полный regression
contour зелёный перед будущим rollout.

## Цель

Зафиксировать и выполнить всю недеплойную часть release readiness перед `128.4-128.7`.

## Scope

1. Проверить, что release checklist синхронизирован с текущим production flow.
2. Проверить наличие и релевантность canonical deploy/backup/smoke/rollback scripts.
3. Подтвердить полный зелёный regression run после хвоста stages `122-129`.
4. Зафиксировать, что server-dependent rollout steps остаются blocked до восстановления сервера.
