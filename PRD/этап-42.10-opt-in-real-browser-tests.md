# PRD: Этап 42.10. Optional real browser tests

## Цель

Добавить opt-in browser tests с реальными credentials, которые:
- не входят в обязательный default suite по смыслу;
- запускаются только по явному env-флагу;
- покрывают реальные browser happy-path для API-key и user-token flow.

## Контракт запуска

- требуется `RUN_REAL_BROWSER_TESTS=1`
- для API-key flow: `TEST_DOMAIN`, `TEST_API_KEY`
- для user-token flow: `TEST_USER_TOKEN_BASE_URL`, `TEST_USER_LOGIN`, `TEST_USER_PASSWORD`

## Критерии готовности

- Есть отдельный marker для real browser tests.
- Без opt-in env tests корректно skip'аются.
- При наличии env выполняются against real contour через локальный web UI.
