# PRD: Этап 42.8. Regression test на cleanup browser accounts

## Контекст

Этап `42.7` вводит reusable cleanup helper для browser tests. Теперь нужно
отдельно доказать, что helper действительно удаляет все связанные сущности,
а не только сам account.

## Цель 42.8

Добавить regression test, который:
- проходит browser flow с созданием account/integration/bearer;
- выполняет cleanup helper;
- проверяет, что в тестовой БД не осталось связанных записей.

## Критерии готовности

- После `cleanup_now()` counts по ключевым таблицам равны нулю.
- Regression test проходит в default suite.
