# PRD Этап 2.6: MCP Prompts

## Цель
Реализовать MCP Prompts — готовые шаблоны запросов для типовых сценариев работы
администратора, врача, финансиста. Промпты помогают LLM составить правильный
вызов инструментов без необходимости вручную указывать параметры.

## Реализация
- Один файл `prompts.py` с регистрацией через `@mcp.prompt`
- Каждый промпт возвращает список сообщений с инструкцией для LLM
- ≤ 250 строк итого

## Промпты

### Администратор
- `daily-schedule` — расписание приёмов на день (параметр: date, doctor_id)
- `find-client` — поиск клиента по имени/телефону
- `client-balance` — баланс клиента (invoices + payments)
- `book-appointment` — запись на приём
- `create-invoice` — создать счёт для клиента
- `doctor-workload` — нагрузка врача за период
- `unconfirmed-appointments` — неподтверждённые записи

### Врач
- `pet-history` — история болезни питомца
- `last-vaccinations` — статус вакцинации
- `add-medical-note` — добавить запись в карту
- `current-inpatients` — пациенты в стационаре
- `pet-invoices` — счета по питомцу
- `pet-full-profile` — полный профиль питомца

### Финансы
- `daily-revenue` — выручка за день
- `unpaid-invoices` — неоплаченные счета
- `popular-services` — топ услуг/товаров

### Склад и клиентская база
- `search-good` — поиск товара/услуги
- `low-stock` — товары с низким остатком
- `new-clients` — новые клиенты за период
- `client-no-visit` — клиенты без визитов 365+ дней

## Критерии готовности
- Все промпты зарегистрированы в server.py
- Каждый промпт содержит понятные инструкции для LLM
